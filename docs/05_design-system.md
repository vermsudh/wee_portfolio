# Portfolio Design System

This document defines the **visual design system** for the portfolio website.

A design system ensures:

* Visual consistency
* Maintainable styling
* Reusable UI patterns
* Faster development

The portfolio will follow a **clean, modern developer aesthetic** using  **neutral tones and minimal UI** .

---

# 1. Design Principles

The overall design should follow these principles:

**Minimalism**

Keep layouts simple and uncluttered.

**Readable Typography**

Content should be easy to read with proper spacing.

**Soft Neutral Colors**

Use warm neutral tones instead of strong colors.

**Subtle Motion**

Animations should enhance the UI but not distract from the content.

**Responsive Layout**

The website should work well on all screen sizes.

---

The portfolio will use a  **warm neutral palette** .

Primary colors are used for backgrounds, text, and UI elements.
